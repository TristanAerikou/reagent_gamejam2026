# reagent_gamejam2026

## Game Objective
1. Je krijgt een lijst van te maken potions.
2. Infinite time om potions te maken.
3. Tussentijds distractions:
    1. verkeerde ingredient valt in de ketel --> je moet een ingrediënt zoeken om het te fixen
    2. Politie komt -> maak invisibility potion, of dood politie met "koffie"
    3. Buren klagen -> maak potion dat als parfum werkt
    4. Sommige potions zijn brandbaar; WATER NODIG

## Potions
- Onzichtbaarheid
- Snelheid
- Instant Death
- Parfum
- etc. 

## Hoe potions maken
1. Potion heeft 3 ingrediënten: zout, kattenpoot, spaghettisliert
2. Ga naar juiste locatie waar het ingrediënt in zit (er zitten meer ingrediënten op 1 locatie)
3. Drop de item's in de ketel.
